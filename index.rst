======================================
Adventuron User Guide
======================================

Welcome To Adventuron
=====================

Adventuron is a TEXT ADVENTURE and GAMEBOOK game creation system.
Adventuron can be used to develop textual games (with optional graphics / multimedia) for desktop, mobile platforms (HTML5), and retro platforms.

AdventuronはTEXT ADVENTUREとGAMEBOOKのゲーム作成システムです。
Adventuron は、デスクトップ、モバイルプラットフォーム（HTML5）、レトロなプラットフォーム向けのテキストゲーム（オプションのグラフィック/マルチメディア付き）を開発するために使用できます。

ZX Spectrum Next + Compatibility Mode
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Adventuron (in 8-bit compatibility mode) can be used to target platforms such as ZX Spectrum, ZX Spectrum Next, Amstrad PCW, Amiga, & more.
If you are interested in this aspect of Adventuron, then please follow Tutorial A.

Adventuron (8ビット互換モード) は、ZX Spectrum, ZX Spectrum Next, Amstrad PCW, Amiga, などのプラットフォームをターゲットとして使用することができます。
興味がある方は、チュートリアルAに従ってください。

Features
~~~~~~~~
* Easy to learn, multiple tutorials, well documented.
* A web-based code-editor with autocompletion.
* Fast development - play as you code.
* Play games on desktop, mobile, and classic computer platforms.
* Export to 8-bit platforms: Amstrad CPC, Commodore 64, ZX Spectrum, ZX Spectrum Next, ..
* A fun way to learn to code.
* Suitable for Raspberry Pi
* Excellent support.


* 学習が簡単で、複数のチュートリアルがあり、ドキュメントも充実しています。
* 自動補完機能を備えたウェブベースのコードエディタ。
* 迅速な開発 - コードを書きながらプレイできます。
* デスクトップ、モバイル、クラシックコンピュータのプラットフォームでゲームをプレイできます。
* 8ビットプラットフォームへのエクスポート。Amstrad CPC、Commodore 64、ZX Spectrum、ZX Spectrum Next、...
* コードを学ぶ楽しい方法
* Raspberry Piに適しています。
* 素晴らしいサポート

Text Adventures
~~~~~~~~~~~~~~~

A text adventure game is a type of game in which the player interacts with via text (similar to instant messaging).
Typically, text adventure games are a blend of storytelling and problem solving, but some adventures may choose to focus on one or the other.
Text adventures can also feature bright vibrant graphics.
Click here to play an interactive tutorial game (5 minutes duration).

テキストアドベンチャーゲームは、プレイヤーがテキスト（インスタントメッセージングに似ている）を介してと対話するゲームの種類です。
一般的に、テキストアドベンチャーゲームは、ストーリーテリングと問題解決のブレンドですが、いくつかの冒険は、どちらか一方に焦点を当てることを選択することができます。
テキストアドベンチャーは、鮮やかで鮮やかなグラフィックを特徴とすることもできます。
